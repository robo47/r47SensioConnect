== Description ==

r47SensioConnect allows you to show profile-information and badges from SensioConnect

== Installation ==

Zip:
Upload the r47SensioConnect plugin to your blog

Activate it
Add the Widgets where you want

Git:
Clone the r47SensioConnect-Plugin from github into the /wp-content/plugin/r47SensioConnect directory

git clone git://github.com/robo47/r47SensioConnect.git wp-content/plugin/r47SensioConnec

Activate it
Add the Widgets where you want
